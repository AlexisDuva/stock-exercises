# stock-exercises

A list of LeetCode exercises I have completed to learn how to code in Go. 

It is possible to check my answers by clicking on the LeetCode page link above a function, copy-pasting the content of the file in the online editor and pressing the "Run" button. It may also be required to copy-paste functions from other files. For example, minSubArrayLen depends on removeDuplicates.

It is also possible to test the functions locally by calling them from main.go.

## Installation

```bash
git clone https://github.com/AlexisDuva/stock-exercises.git
cd stock-exercises
go build ./...
```

## Usage

```bash
go run ./...
```
